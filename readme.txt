FISH FRESHNESS PREDICTION APPLICATION


======================================

# Requirements :

- Python installed , with path added to the environment variables

- An Internet connection

- Turn off the popup Blocker

--------------------------------------

Steps to run the application :

1. Unziped the file (if not)

2. run the start.sh bash file

	- the dependencies will get install automatically

3. after everything is installed , click on the localhost link

4. the web page will open in the browser
